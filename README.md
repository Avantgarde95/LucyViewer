# LucyViewer

Simple Embree ray tracing demo: Lucy 3D model viewer

### Requirements

- [CMake](https://cmake.org/)

### Dependencies

- [Qt](https://www.qt.io/) (Recommended version: 5.13.1)
- [Intel Embree](https://www.embree.org/) (Recommended version: 3.11.0)
